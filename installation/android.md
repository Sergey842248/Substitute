# Android

### Android (Native App)

1. Head to the [GitHub Releases](https://github.com/Sergey842248/Substitute/releases) Page
2. Download the latest `Substitute-Android.apk` file.

<div align="left"><figure><img src="../.gitbook/assets/Download.png" alt="" width="188"><figcaption></figcaption></figure></div>

3. Open the downloaded APK file on your Android device to begin the installation process. You may need to enable installation from unknown sources in your device settings.

<div><figure><img src="../.gitbook/assets/Install-Error.png" alt="" width="375"><figcaption><p>After opening the .apk file click<br>on Settings</p></figcaption></figure> <figure><img src="../.gitbook/assets/Install-Permission.png" alt="" width="375"><figcaption><p>Enable the toggle<br>"Allow from this Source"</p></figcaption></figure> <figure><img src="../.gitbook/assets/Installing.png" alt="" width="375"><figcaption><p>In the new popup dialog<br>Click "Install"</p></figcaption></figure></div>
