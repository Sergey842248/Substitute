# Android

1. Head to the [GitHub Releases](https://github.com/Sergey842248/Substitute/releases) Page
2. Download the latest `Substitute-Android.apk` file

<figure><img src="../.gitbook/assets/Download.png" alt="" width="188"><figcaption></figcaption></figure>



3. Open the downloaded APK file on your Android device to begin the installation process. You may need to enable installation from unknown sources in your device settings.

<div><figure><img src="../.gitbook/assets/Install-Error (1).png" alt="" width="188"><figcaption><p>After opening the .apk file click<br>on Settings</p></figcaption></figure> <figure><img src="../.gitbook/assets/Install-Permission (1).png" alt="" width="188"><figcaption><p>Enable the toggle <br>"Allow from this Source"</p></figcaption></figure> <figure><img src="../.gitbook/assets/Installing.png" alt="" width="188"><figcaption><p>In the new popup dialog<br>Click "Install"</p></figcaption></figure></div>
