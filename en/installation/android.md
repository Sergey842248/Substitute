# Android

{% stepper %}
{% step %}
### Download the APK

Head to the [GitHub Releases](https://github.com/Sergey842248/Substitute/releases) Page and download the latest `Substitute-Android.apk` file.

<div align="left"><figure><img src="../../.gitbook/assets/Download.png" alt="" width="188"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
### Install the APK on your Android device

Open the downloaded APK file. You may need to enable installation from unknown sources in your device settings.

<div align="left"><figure><img src="../../.gitbook/assets/Install-Error.png" alt="" width="188"><figcaption><p>After opening the .apk file click<br>on Settings</p></figcaption></figure> <figure><img src="../../.gitbook/assets/Install-Permission.png" alt="" width="188"><figcaption><p>Enable the toggle<br>"Allow from this Source"</p></figcaption></figure> <figure><img src="../../.gitbook/assets/Installing.png" alt="" width="188"><figcaption><p>In the new popup dialog<br>Click "Install"</p></figcaption></figure></div>
{% endstep %}
{% endstepper %}
